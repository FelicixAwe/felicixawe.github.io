Basics Introduction#

The whole website is developed based on Hugo and PaperMod theme.

Deployment#

The website is deployed on GitHub Page.

  1. Create a new repository with the name yourname.github.io

  2. Initialize a git in public folder.

    1
2
3
4
5
6

    cd public
git init
git remote add origin [repository url]
git add .
git commit -m 'init'
git push -f --set-upstream origin master

  3. hugo -F --cleanDestinationDir will generate the output file under public folder. Push changes in public folder to Github, and website will be automatically deployed.

Code font#

  1. Content Fonts

Here I use Josephin Sans. Copy the generated HTML and CSS, insert the HTML into themes/PaperMod/layouts/partials/extend_head.html, and insert CSS into themes/PaperMod/layouts/assets/css/extended/blank.css

  1. Code Fonts

I use Ubuntu Mono. Same steps as content fonts.
